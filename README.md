# 🛒 Система управления интернет-магазином (PHP/MySQL)

Полноценное веб-приложение для управления товарами, категориями, пользователями и заказами. Включает административную панель, личный кабинет пользователя, корзину и систему заказов.

## 📋 Возможности

### 👥 Пользователи
- Регистрация и авторизация (с хешированием паролей `password_hash`)
- Загрузка аватарки (JPG, PNG, GIF, WEBP до 2MB)
- Личный кабинет с историей заказов
- Администратор по умолчанию: `admin` / `admin123`

### 🛍️ Магазин (`shop.php`)
- Просмотр всех товаров с изображениями
- Добавление товаров в корзину (сессионная корзина)
- Изменение количества товаров в корзине
- Оформление заказа (только для авторизованных пользователей)
- Автоматическое списание товара со склада

### 👑 Административная панель (`deda.php`)
- **Управление товарами:** добавление, редактирование, удаление (по одному или массово)
- **Управление категориями:** создание, удаление (с автоматической расчисткой связей)
- **Фильтрация товаров:** по категории, поиск по названию
- **Сортировка товаров:** по возрастанию/убыванию цены
- **Загрузка изображений** для товаров (в папку `uploads/`)
- Смена аватарки администратора

### 📦 Заказы
- Автоматическое создание заказа с уникальным номером
- Запись в таблицу `orders`
- Обновление остатков товаров после оформления
- История заказов в личном кабинете пользователя

### 🗄️ База данных
- Таблицы: `users`, `tovars`, `categories`, `orders`
- Поддержка внешних ключей (категории – товары)
- Автоматическое создание таблиц при первом запуске

## 🛠️ Технологии
- PHP 7.4+ (процедурный стиль, сессии, работа с файлами)
- MySQL (MySQLi)
- HTML/CSS (адаптивный дизайн)
- JavaScript (предпросмотр изображений, массовый выбор)
